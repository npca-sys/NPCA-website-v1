このサイトのこととか

このサイトは、見る人に優しいサイトを作ろうと心に決めた人が作ったものです。HTML 4.01 Strictを用い、日本語ベースで書かれています。

今回の改良点は、

  • テキストブラウザ / 印刷に対応(*1)
  • タグの論理構造見直し(<div>や<span>大幅削減 等)
  • ユーザビリティに配慮した文章構成に変更

となっております。

また、動作確認を行ったブラウザは以下の通りです。

  • Internet Explorer 6.02
  • Netscape Navigator 7.1 / 8.0 Beta
  • Opera 8.50
  • FireFox 1.07
  • Lynx 2.85rel(テキストブラウザ)

*1: 現状、印刷に対応したのはトップページのみです。
